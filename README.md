# Trabajo Práctico de Teoría del Lenguaje

### Para compilar y correr el servidor, utilizamos el siguiente comando (Antes ubicándonos en el directorio del serverPackage):
```
go run server.go
```

### Por su parte, utilizaremos el siguiente comando para compilar y correr a los respectivos clientes en una nueva terminal (Antes ubicándonos en el directorio de clientPackage):
```
go run client.go
```
